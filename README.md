# UED VSTEP Bậc 3 – gói sẵn sàng cho GitHub Pages

Các tệp trong thư mục này đã được biên dịch sẵn. Không cần cài Node.js và không chạy lệnh build.

## Cách tải lên thủ công

1. Giải nén tệp ZIP.
2. Mở kho `ued-vstep-bac-3` trên GitHub.
3. Chọn **Add file → Upload files**.
4. Kéo toàn bộ **nội dung bên trong thư mục đã giải nén** lên kho. Phải nhìn thấy `index.html`, `404.html`, `.nojekyll`, `assets` và `README.md` ngay ở thư mục gốc.
5. Chọn **Commit changes**.
6. Mở **Settings → Pages**.
7. Tại **Build and deployment**, chọn **Deploy from a branch**.
8. Chọn nhánh **main**, thư mục **/(root)** rồi nhấn **Save**.

Sau khi GitHub hoàn tất xuất bản, địa chỉ dự kiến là:

`https://nguyendangtrunghieu81-rgb.github.io/ued-vstep-bac-3/`

## Lưu ý

- Không tải nguyên tệp ZIP lên GitHub; GitHub Pages không tự giải nén.
- Không đổi tên thư mục `assets` hoặc các tệp bên trong.
- Website lưu tiến độ trong trình duyệt của từng người dùng.
- Ghi âm cần HTTPS; GitHub Pages đáp ứng điều kiện này sau khi xuất bản.
- Phần Nghe tự ưu tiên giọng tiếng Anh tự nhiên nhất mà thiết bị có và cho phép đổi giọng/tốc độ; chất lượng có thể khác nhau giữa Windows, macOS, Android và iOS.
- Đây là công cụ luyện tập tham khảo, không phải hệ thống thi chính thức.
